# Car Damage Prediction

An end-to-end **Computer Vision & Deep Learning** project for detecting and predicting car damage from images. This system enables automated vehicle damage assessment using trained machine learning models, a backend API, and a user-friendly web interface.


## Project Overview

Car Damage Prediction is designed to automate the process of identifying and classifying vehicle damage from images.

### Objectives

- Detect damaged areas in vehicle images.
- Classify severity or type of damage.
- Provide a simple interface for users to upload images.
- Serve predictions through a scalable backend API.

### Real-World Applications

- Insurance claim automation
- Vehicle inspection systems
- Repair cost estimation
- Fleet management solutions

## Features

-  Image upload support
-  Deep learning-based prediction
-  FastAPI backend for real-time inference
-  Streamlit web interface
